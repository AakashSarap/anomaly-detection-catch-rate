# 📊 Anomaly Detection on Catch Rate (YouTube T&S)

## Overview
This project simulates daily catch rate metrics and detects anomalies using statistical thresholds.

## Approach
- Generated synthetic data (Normal distribution)
- Injected anomalies
- Applied 2 standard deviation rule
- Visualized results

## Tech Stack
- Python
- NumPy
- Matplotlib

## Results
Successfully identified anomaly days using statistical thresholds.
